## 🐛 Bug 清单

### BUS-001 [P2] — `read()` limit=0 不返回空列表

- **复现**：`read('project', limit=0)` → 返回全部事件（期望返回 `[]`）
- **根因**：Python 切片 `entries[-0:]` 等价于 `entries[0:]`，因为 `-0 == 0`
- **文件**：`glink/bus/main_bus.py` → `return entries[-limit:]`
- **修复**：在 `read()` 开头加 `if limit == 0: return []`
- **影响范围**：所有调用 `read()` 且可能传 `limit=0` 的场景
- **状态**：⚠️ 未修复

### BUS-002 [P1] — `bus_path()` 路径穿越漏洞

- **根因**：`project_name` 直接拼接路径，未做 sanitization
- **文件**：`glink/bus/main_bus.py` → `bus_path()` 函数
- **修复**：对 `project_name` 做白名单过滤 `re.sub(r'[^a-zA-Z0-9_-]', '_', project_name)`
- **影响范围**：理论上可写入用户目录下任意位置（需 OS 权限）
- **状态**：⚠️ 未修复
